开发必备工具

发布于：2020-05-20 19：00 | 阅读：10000 | 标签 #关于 #关于
Git部分：git是一个用于管理版本控制的非常优秀的工具，使用它可以实现很多诸如创建分支代码、回滚、合并代码等的操作，它可以跟个人github账号仓库（完全免费）绑定达到类似创建云盘的效果。
初次安装后：1.打开cmd命令行，2.输入“git config --global user.name (用户名)”和“git config --global user.email (用户邮箱)”，可以通过"git config --list"检查信息清单
创建项目：进入cmd找到相关目录或者使用编辑器的terminal终端输入“git init”
